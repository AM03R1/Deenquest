# DeenQuest

DeenQuest is een mobile-first PWA MVP voor jongeren die spelenderwijs meer willen leren over Islam met doelen, quizzen, XP en levels.

## Functionaliteiten

- Dashboard met level, XP en voortgang
- Persoonlijke doelen toevoegen en afstrepen
- Quiz met 10 meerkeuzevragen
- XP en level-up systeem
- Progressie en quizgeschiedenis in localStorage
- Dark mode en demo-reset via instellingen
- PWA manifest en service worker

## Lokaal openen

Start een simpele lokale server:

```bash
python3 -m http.server 4173
```

Open daarna:

```text
http://127.0.0.1:4173/
```
